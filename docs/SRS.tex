\section{Introduction}
\subsection{Purpose}
Define the requirements for an AI system that predicts air pollution and displays results in a dashboard.

\subsection{Intended Audience}
Professor, developers, testers, data scientists.

\subsection{Overview}
This SRS describes functional and non-functional requirements for the system.

\section{External Interface Requirements}
\subsection{User Interface}
\begin{itemize}
    \item Dashboard with charts and graphs
    \item Prediction module
    \item Data upload/input page
\end{itemize}

\subsection{Software Interfaces}
\begin{itemize}
    \item Python environment
    \item Pandas, NumPy, Matplotlib, Scikit-Learn
    \item Streamlit or Flask backend
\end{itemize}

\section{Legal and Ethical Considerations}
\begin{itemize}
    \item Data privacy for uploaded datasets
    \item No harmful misuse of predictions
    \item Transparency of AI model accuracy
\end{itemize}

\section{Glossary}
\begin{itemize}
    \item API - Application Programming Interface
    \item PM10 - Coarse particulate matter
    \item UI - User Interface
\end{itemize}
